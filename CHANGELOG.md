# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-06-04

### Added
- ✨ Initial release of Zac Personal Assistant
- 🎤 Voice control with Vosk speech-to-text (Portuguese)
- 🔊 Text-to-speech synthesis using pyttsx3
- 🌐 Browser automation with Playwright
- 📝 Task management with SQLite
- 📅 Calendar/Events management
- 💾 Personal memory/knowledge base
- 📊 Google Sheets integration (free API)
- 🔄 Scheduler for recurring tasks
- 🚀 REST API with FastAPI
- 📦 Modular architecture
- ✅ Comprehensive tests
- 📖 Complete documentation
- 🛠️ Automated installation script

### Features
- Command-based interface
- Voice recognition in Portuguese
- Task prioritization and tagging
- Calendar reminders
- Browser control (open, navigate, click, fill forms)
- Google Sheets integration (read/write)
- Local database with SQLite
- API for remote control
- Scheduler with recurring tasks
- Memory management (people, goals, preferences)

### Technology Stack
- Python 3.12
- FastAPI
- SQLite
- Playwright
- Vosk
- pyttsx3
- Google Sheets API
- Pydantic
